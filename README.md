# Podcast Gerado com IA

Este projeto foi desenvolvido para o desafio da DIO de Inteligência Artificial Generativa.  
O objetivo é criar um episódio de podcast usando prompts de IA, gerar o roteiro e produzir o áudio final.

## Estrutura do Repositório

- **prompts/** → contém o prompt usado para gerar o episódio (`prompts.txt`)  
- **roteiro/** → contém o roteiro gerado pela IA (`episodio_01.txt`)  
- **audio/** → contém o episódio final em áudio (`episodio_01.mp3`)  
- **README.md** → descrição do projeto

## Tema do Episódio 01
**Tecnologia e Emoções**

O episódio explora como a tecnologia influencia nossas emoções e a importância de atenção e presença no mundo digital.

## Ferramentas Usadas
- ChatGPT → criação do prompt e roteiro  
- ElevenLabs → geração da voz em áudio  
- GitHub → versionamento do projeto

## Como Ouvir
Baixe ou abra o arquivo `/audio/episodio_01.mp3` e ouça o episódio.

## Observações
Este projeto é uma demonstração prática de uso de IA generativa aplicada a podcast, visando aprendizado e portfólio.
